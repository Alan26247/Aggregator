Задание:
	Нужно написать REST API агрегатор новостей.
	Пользователь подает на вход адрес новостного сайта или его RSS-ленты
	База данных агрегатора начинает автоматически пополняться новостями с этого сайта.
	У пользователя есть возможность просматривать список новостей из базы и искать их 
	по подстроке в заголовке новости.
	В качестве примера требуется подключить 2 любых новостных сайта на выбор.
	Результат - исходный код агрегатора, а также рабочие адреса.
	Язык C# .Net. Хранилище - любая реляционная база.


Решение:

	Результат работы приложения можно проверить по адресу:	http://rsgservice.ru/index.html

	При решении данной задачи был создан Web Api - Asp Net Core проект. В качестве 
реляционной базы данных использована MySql. Для доступа к базе использован Entity Framework.
В базе данных я создал две таблицы. Первая Channels - хранит список использованных ссылок 
при запросах. Вторая News - хранит соответственно весь список новостей.

Рабочие ссылки:
		https://yandex.ru/company/press_releases/news.rss    -    яндекс новости
		
		https://news.rambler.ru/rss/Stavropol/ 		-     новости ставрополь

REST API:
		POST:	/api/aggregator    -			Обновить данные. В теле запроса
								необходимо передать параметр
								RssUrl типа string. Сохраняет 
								результат поиска в базу данных.

		GET:	/api/aggregator	  -			Получить весь список новостей 
								из базы данных в формате JSON

		GET:	/api/aggregator/{findStr}	  -	Получить список новостей из 
								базы данных с фильтрацией в заголовке
								по findStr в формате JSON

		DELETE:	/api/aggregator	  -			Очистить базу данных
